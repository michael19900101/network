步骤1：添加Retrofit库的依赖
1.在 Gradle加入Retrofit库的依赖
build.gradle
dependencies {
    compile 'com.squareup.retrofit2:retrofit:$retrofit_version'
}

2.添加网络权限
AndroidManifest.xml
<uses-permission android:name="android.permission.INTERNET"/>
